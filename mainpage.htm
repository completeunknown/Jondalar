    An input box with a datalist. This HTML5 feature is implemented differently in the various browsers.

    Firefox v7.0.1

    The list appears when you click a second time after the field gets the focus.
    As long as the field has the focus, each mouse click toggles the list on and off.
    Typing a string in the box brings up the words in the list that CONTAIN that string.

    Opera v11.52

    The list appears as soon as the field gets the focus, and disappears if you click it again.
    It doesn't reappear until it looses and regains focus.
    Typing a string int he box brings up only those words in the list that START with that string.

    The datalist element is not supported in Internet Explorer v9, Google v15.0.874.106 and Safari v5.0.5.
